# Goal

The program must output the given map, making the # fall to the bottom of the grid.

The map is composed of '.' and '#'.


Input  
	...#...#.#.#...#.
	
	.#..#...#....#...
	
	..........#......
	
	..###...###..##..
	
	#################

Output  
	.................
	
	.................
	
	...##...###..#...
	
	.####..#####.###.
	
	#################
